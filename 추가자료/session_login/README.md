# Getting Started
## Backend Setup
### 1 ) Navigate to the server directory
```bash
cd server
```
### 2 ) Install Dependencies
To install all necessary libraries and dependencies, run:
```bash
npm install
```

### 3 ) Running the Node.js Server
Start the Node.js server:
```bash
npm start
```

<br/>
